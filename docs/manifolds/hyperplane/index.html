1.4 HyperPlanes

1.4.1 Bijection​

Recall that if we have a function fff that maps from set AAA to BBB, then we could desribe as following:

  • Not a map: ∃x∈A\exist x \in A∃x∈A, such that xxx points to many y∈By \in By∈B.
  • Injective: ∀x1,x2∈A\forall x_1, x_2 \in A∀x1​,x2​∈A, f(x1)≠f(x2)  ⟺  x1≠x2f(x_1) \neq f(x_2) \iff x_1 \neq x_2f(x1​)=f(x2​)⟺x1​=x2​.
  • Surjective: ∀y∈B\forall y \in B∀y∈B, we can find some x∈Ax \in Ax∈A such that f(x)=yf(x) = yf(x)=y.
  • Bijective: both injective and surjective.

That being said, a function f:A→Bf:A\to Bf:A→B is bijective if and only if it has an inverse function g:B→Ag: B \to Ag:B→A.

If we have a function f:A→Bf: A \to Bf:A→B that's bijective, then AAA is global euclidean.

1.4.2 Hyperplane​

Definition of hyperplane

A hyperplane is a subplace that is one less dimension than the space around it.

For example, a hyperplane of a 3-dimensional space is a 2-dimensional plane.

1.4.3 Cordinate Hyperplane​

Definition of cordinate hyperplane

We define the j-th cordinate hyperplace of Rn+1\mathbb{R}^{n+1}Rn+1 as following:

Cj=X∈Rn+1,Xj=0\mathbb{C}_{j} = {X \in \mathbb{R}^{n+1}, X^j = 0}Cj​=X∈Rn+1,Xj=0

where 1≤j≤n+11 \leq j \leq n+11≤j≤n+1

Note we have a linear mapping fff from Cj\mathbb{C}_jCj​ to Rn\mathbb{R}^nRn:

f:Cj→Rn=(X1,…,Xj−1,Xj+1,…,Xn+1)f: \mathbb{C}_j \to \mathbb{R}^n = (X^1,\dots,X^{j-1},X^{j+1},\dots,X^{n+1})f:Cj​→Rn=(X1,…,Xj−1,Xj+1,…,Xn+1)

and its reverse function ggg:

g:Rn→Cj=(X1,…,Xj−1,0,Xj+1,…,Xn+1)g: \mathbb{R}^n \to \mathbb{C}_j = (X^1,\dots,X^{j-1},0,X^{j+1},\dots,X^{n+1})g:Rn→Cj​=(X1,…,Xj−1,0,Xj+1,…,Xn+1)

We conclude that fff is bijective and thus the j-th cordinate hyperplace is globally euclidean.

1.4.4 Affine Hyperplane​

Definition of affine hyperplane

An affine hyperplane in Rn+1\mathbb{R}^{n+1}Rn+1 is defined as:

A={X∈Rn+1,X⋅N=a}\mathbb{A} = \{X \in \mathbb{R}^{n+1}, X \cdot N = a \}A={X∈Rn+1,X⋅N=a}

where a∈Ra \in \mathbb{R}a∈R and NNN is called a norm to A\mathbb{A}A, a fixed non-zero vector.
